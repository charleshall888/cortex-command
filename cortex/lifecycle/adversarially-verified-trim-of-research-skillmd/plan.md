# Plan: adversarially-verified-trim-of-research-skillmd

## Overview
Trim `skills/research/SKILL.md` by running a role-separated verification pipeline — a fresh auditor proposes classified cuts, a *distinct* verifier discharges each against a citable surviving anchor, the orchestrator applies only the verified-safe subset, and a *blind fresh oracle* (post-trim file only) re-answers a 9-item behavioral battery (the spec's 7 plus 2 coverage items) that, together with a deterministic keep-verbatim preserve-list, bounds behavior-neutrality from below — then apply three lifecycle/refine rider canonicalizations while the toolchain is warm. Evidence is recorded as a precedent-shaped `trims_verified` entry plus a `behavioral_equivalence` battery and a `provenance` block (pinned blob SHAs + dispatched agent IDs) in `evidence.json`.
**Architectural Pattern**: pipeline
<!-- propose → discharge → apply → integrity-gate / blind-oracle → reconcile; each stage gates the next, with role-separated agents preventing self-certification. -->

## Outline

### Phase 1: Verified-safe trim of research/SKILL.md (tasks: 1, 2, 3, 4, 5, 6, 7, 8)
**Goal**: Apply the classification-driven, independently-verified, behavior-neutral trim to the canonical file + mirror, recording all evidence.
**Checkpoint**: `skills/research/SKILL.md` < 15018 B with all five `## Step` headings intact, all five fenced prompt-block bodies byte-identical (diffed against the pinned pre-trim blob), all 9 behavioral-battery answers matching baseline, `evidence.json` complete (incl. `provenance` blob SHAs + distinct agent IDs), mirror byte-identical, Phase-1 commit landed.

### Phase 2: Rider micro-canonicalizations (tasks: 9, 10, 11, 12)
**Goal**: Convert three inline-duplicated rules to pointers at their canonical homes, retargeting every dependent back-reference in the same change (R9 may decline with recorded rationale).
**Checkpoint**: Each rider applied (or R9 declined with rationale), `just test` green, `cortex-check-skill-path` reports 0 violations, all mirrors byte-identical, Phase-2 commit landed.
