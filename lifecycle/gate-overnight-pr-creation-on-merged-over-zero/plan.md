# Plan: gate-overnight-pr-creation-on-merged-over-zero

## Overview

Gate the home-repo integration-branch PR on `MC_MERGED_COUNT` in `claude/overnight/runner.sh`: zero → `--draft` with `[ZERO PROGRESS]` title; non-zero → existing behavior. Nine tasks, in three phases: (1) `--dry-run` scaffolding + fixtures so later tasks are testable under subprocess capture; (2) runner.sh behavior changes (gate, pre-check, resume-flow state flip with once-per-PR marker, breadcrumb comments); (3) consumer-side walkthrough update + pytest regression suite + live-path non-regression check.
