The linter parses this file at load time. Schema (markdown table, 5 columns):

| Column | Constraint |
| --- | --- |
| `script` | Matches regex `cortex-[a-z][a-z0-9-]*`. Bare token, no `bin/` prefix. |
| `category` | One of `maintainer-only-tool`, `library-internal`, `deprecated-pending-removal`. |
| `rationale` | ≥30 characters after trim. Forbidden literals (case-insensitive): `internal`, `misc`, `tbd`, `n/a`, `pending`, `temporary`. Be specific. |
| `lifecycle_id` | Lifecycle ticket / backlog id that introduced the entry. |
| `added_date` | `YYYY-MM-DD`. |
